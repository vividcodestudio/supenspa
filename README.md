## 🚨 Troubleshooting

**Pages show "Not Found"**
- Check file names match exactly
- Ensure all files uploaded

**WhatsApp buttons don't work**
- Update phone number in search/replace
- Test in browser first (opens web.whatsapp.com)
- On mobile, opens WhatsApp app

**Images missing**
- Verify `assets/images/` uploaded
- Check image file paths in HTML

**Styling looks broken**
- Ensure `assets/css/` folder uploaded
- Clear browser cache (Cmd+Shift+R)
- Check file permissions
