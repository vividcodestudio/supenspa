## 🟢 OPTION 1: NETLIFY (EASIEST - 5 MINUTES)

### Step 1: Create GitHub Account
Go to **github.com** → Sign up (free)

### Step 2: Create New Repository
- Click **New Repository**
- Name: `supenspa-website`
- Click **Create Repository**

### Step 3: Upload Your Files
- Click **Add files** → **Upload files**
- Select all files from `/website-pages/` folder
- Click **Commit changes**

### Step 4: Deploy to Netlify
1. Go to **netlify.com**
2. Click **Sign up** (use your GitHub account)
3. Click **New site from Git**
4. Select **GitHub** and your new repository
5. Click **Deploy site**

**Done!** Your site is live in 2-3 minutes  
**URL:** `your-site-name.netlify.app`

### Add Custom Domain (Optional)
1. Buy domain from **GoDaddy** or **Namecheap** (~$10/year)
2. In Netlify: **Site settings** → **Domain** → **Add custom domain**
3. Follow instructions to point domain
